# RetroVideo Manager

Proyecto semestral desarrollado para la asignatura Desarrollo Fullstack I.

## Descripción
RetroVideo Manager es un sistema basado en arquitectura de microservicios para la gestión integral de una tienda de cintas VHS.

## Integrantes
- Fernanda Poblete
- Álvaro Vásquez
- Rafael Gallardo

## Microservicios
- MS-Autenticación
- MS-Usuarios
- MS-Catálogo-Películas
- MS-Inventario-Físico
- MS-Alquileres
- MS-Ventas
- MS-Membresías
- MS-Taller-Reparación
- MS-Notificaciones
- MS-Pagos

## Tecnologías utilizadas
- Java 21
- Spring Boot
- Maven
- MySQL
- GitHub
- Postman

## Arquitectura
Sistema distribuido en 10 microservicios independientes, cada uno con su propia base de datos y responsabilidades específicas.
